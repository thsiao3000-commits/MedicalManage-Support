常見問題

如何使用 AI 辨識藥品？

首先在「設定」中輸入您的 OpenAI API Key，然後在新增藥品時點選相機圖示，拍攝藥品照片即可自動辨識。您可以在設定中選擇 GPT-4o 或 GPT-4o Mini 模型。

什麼是 OpenAI API Key？如何取得？

OpenAI API Key 是使用 AI 辨識功能所需的金鑰。請前往 platform.openai.com/api-keys 註冊並取得。使用 AI 辨識功能會依照 OpenAI 的定價產生少量費用。

不使用 AI 辨識也可以嗎？

可以！AI 辨識是選用功能。您可以手動輸入所有藥品資訊，App 的其他功能（提醒、效期管理、用藥紀錄等）完全不需要 API Key。

我的資料安全嗎？

您的資料儲存於您本人的 iCloud（透過 Apple CloudKit），不經過本 App 任何伺服器，可在登入相同 Apple ID 的多裝置自動同步。API Key 以 iOS 鑰匙圈加密保存。只有在您主動使用 AI 辨識時，藥品照片才會直接從您的裝置傳送至 OpenAI 或 Groq，開發者不接觸這些資料。詳見 隱私權政策。

如何備份或轉移資料？

前往「設定」頁面，使用匯出功能可將所有藥品資料匯出為 .medbox 檔案。您可以選擇是否包含照片。透過 AirDrop、郵件或雲端硬碟分享，在新裝置上匯入即可還原。

為什麼收不到用藥提醒通知？

請確認：(1) 已允許 App 傳送通知（設定 > 通知 > 家庭藥箱）；(2) 提醒已設為「啟用」狀態；(3) 裝置未開啟勿擾模式。如仍有問題，可至 App 設定中點選「重新排程所有提醒」。

支援哪些 iPhone 機型？

支援 iOS 18.1 或以上版本的所有 iPhone 機型。
